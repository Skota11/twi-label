ツイートラベルを復活させます。  
使い方は、上の３つのファイルをダウンロードし、任意のフォルダーを作成し、その中に入れます。  

Chromeの場合は、`chrome://extensions`  
Edgeの場合は、`edge://extensions/`  
Operaの場合は、`opera://extensions/`  

上のURLにアクセスします。
